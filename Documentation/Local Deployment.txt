Concise Instructions for Local Deployment


Prerequisites:
- .NET Core 6.0 (https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-6.0.428-windows-x64-installer)
- Git (https://git-scm.com/downloads)
- Docker Desktop (optional)
- Visual Studio (optional)


Step 1: Get source code
- Clone the repo using `git clone https://github.com/Christopher-Powers/BucStop-Goofin.git`

Step 2: Run application
- Navigate to Bucstop-Goofin/BucStop WebApp/BucStop/ in the command line interface and run `dotnet run`  OR
